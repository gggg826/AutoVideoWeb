# 启动脚本使用说明

本项目提供了三种启动方式，请根据您的系统选择：

## 方式 1: Windows 批处理文件 (推荐 - 简单但英文界面)

```cmd
start.bat
```

- **优点**: 简单易用，双击即可运行
- **缺点**: 界面为英文（Windows CMD 对中文支持不好）
- **适用**: 所有 Windows 系统

## 方式 2: PowerShell 脚本 (推荐 - 中文界面)

```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

或者在 PowerShell 中直接运行：
```powershell
.\start.ps1
```

- **优点**: 中文界面，彩色输出，更美观
- **缺点**: 可能需要设置执行策略
- **适用**: Windows 10/11 (推荐使用)

**如果遇到权限问题**，以管理员身份运行 PowerShell 并执行：
```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 方式 3: Linux/Mac 系统

```bash
chmod +x start.sh
./start.sh
```

- **优点**: 完整的 Unix/Linux 支持
- **适用**: Linux, macOS, WSL

## 启动后访问

服务器启动后，可以通过以下地址访问：

- **主页**: http://localhost:8000/
- **测试页面**: http://localhost:8000/public/index.html
- **管理后台**: http://localhost:8000/admin/index.html
- **API 文档**: http://localhost:8000/docs

## 停止服务器

在终端/命令行窗口中按 `Ctrl+C` 即可停止服务器。

## 手动启动方式

如果启动脚本���法使用，可以手动执行以下步骤：

```bash
# 1. 创建虚拟环境（首次运行）
cd backend
python -m venv .venv
cd ..

# 2. 激活虚拟环境
# Windows CMD:
backend\.venv\Scripts\activate.bat
# Windows PowerShell:
backend\.venv\Scripts\Activate.ps1
# Linux/Mac:
source backend/.venv/bin/activate

# 3. 安装依赖（首次运行）
pip install -r backend/requirements.txt

# 4. 初始化数据库（首次运行）
python backend/scripts/init_db.py

# 5. 启动服务器
python run.py
```

## 系统要求

- Python 3.10 或更高版本
- Windows 10/11, Linux, 或 macOS
- 约 100MB 磁盘空间（包含依赖包）

## 故障排除

### 问题 1: Python 未找到
**解决**: 安装 Python 3.10+ 并确保添加到 PATH
下载地址: https://www.python.org/downloads/

### 问题 2: PowerShell 执行策略错误
**解决**: 运行 `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`

### 问题 3: 端口 8000 被占用
**解决**: 在 `.env` 文件中修改 `PORT=8001`

### 问题 4: 依赖���装失败
**解决**:
- 检查网络连接
- 使用国内镜像: `pip install -r backend/requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`
